# playground

Small side projects and experiments, each a self-contained static page.

**Live:** https://dipeshrajkarki.github.io/playground/

## Adding a new project

1. Create a folder: `mkdir <project-name>`
2. Drop an `index.html` inside it.
3. Add a link card to the root `index.html`.
4. `git add . && git commit -m "add <project-name>" && git push`

GitHub Pages redeploys automatically on push to `main`.
